# Чат-бот с поддержкой RAG-пайплайна

## Что умеет? (будет уметь)
- Возможность общения в режиме обычного чат бота (как ChatCPT)
- Возможность загрузить свои документы с информацией, по которой хотите сделать вопрос.
Например можно загрузить методичку по учебе и задавать по ней вопросы. Помогает работать с локальной информацией.
- Возможность создавать свои базы знаний, делиться ими с другими пользователями
- Расширение для мудла и канваса для помощи с тестами
